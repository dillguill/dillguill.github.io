# Portfolio Website

This is my personal portfolio website showcasing my projects and skills.

## Features

- Responsive design
- Smooth animations with Framer Motion
- MDX-powered project pages
- Dark/light theme support

## Technologies Used

- React 18
- Tailwind CSS
- Framer Motion
- React Router
- MDX for content
