Portfolio Website

A responsive portfolio website built with Next.js and Tailwind CSS, featuring smooth animations with Framer Motion and MDX-powered project pages.

Features

  • Responsive design
  • Smooth animations with Framer Motion
  • MDX-powered project pages
  • Dark/light theme support

Technologies Used

  • Next.js 16 (React framework with static export)
  • Tailwind CSS
  • Framer Motion
  • MDX for content management
  • next-mdx-remote for dynamic rendering

Getting Started

Clone the repository and install dependencies:

git clone https://github.com/dillguill/portfolio_v04.git
cd portfolio_v04
npm install
npm run dev

For production build:

npm run build
npm run export
