Homelab

Self-hosted infrastructure for AI/ML experimentation, media, and development — no third-party cloud dependency for core services. Built around privacy, data ownership, and developer productivity.

Architecture Overview

Services run containerized via Docker, managed through Portainer. Tailscale provides a WireGuard-based VPN mesh for secure remote access, with Mullvad's native exit-node integration handling protected general browsing egress. Caddy reverse-proxies HTTPS traffic to each service under custom .internal domains. A unified Homepage.dev dashboard provides a single pane of glass across the stack. AI inference runs via Ollama and llama.cpp, unified behind Open WebUI as the primary chat and tooling interface.

AI / ML

Open WebUI

Self-hosted AI chat interface — the primary frontend for local LLM interaction, connecting to multiple backends simultaneously:

  • Ollama API — serves quantized open-weight models for general chat, coding assistants, and tool-calling workflows
  • llama.cpp API — serves GGUF-format models with fine-grained control over context length, quantization, and memory allocation
  • RAG pipeline — ingests the Obsidian vault (notes, references, processed documents), chunking and embedding content into a vector store for context-aware queries against personal knowledge, directly from chat
  • Kokoro — local text-to-speech, layered on for voice interaction
  • Extended with custom skills for repeatable workflows

Whisper API

Local speech-to-text, integrated into Obsidian for voice-based note capture.

Coding agents

OpenCode, Claude Code, and Pi integrated via the Ollama API — terminal-based coding assistance and conversational reasoning against locally served models rather than cloud APIs.

Vane (formerly Perplexica) + SearXNG

Self-hosted, privacy-respecting search: SearXNG aggregates results, Vane synthesizes them through a local model into cited, summarized answers — keeping AI-assisted research entirely local and private.

ComfyUI

Node-based diffusion model interface, run as a native install (not containerized) for direct GPU access and custom node development. Workflows are graph-based and stored as JSON for reproducible, version-controlled generation pipelines.

Networking & Security

  • Tailscale — WireGuard-based VPN mesh connecting all devices; local access plus Mullvad's native exit-node integration for protected general browsing when off-network. No ports exposed to the public internet.
  • Caddy — reverse proxy handling HTTPS routing and automatic certificate management for every service, addressed via custom .internal domains rather than raw ports or IPs.
  • Portainer — container management and monitoring across the Docker fleet.

Knowledge Management

Obsidian

Central knowledge base — notes, research, and documentation stored as plain Markdown, version-controlled via Gitea. Vault content feeds Open WebUI's RAG pipeline, closing the loop between personal research and AI-assisted workflows; Whisper integration allows voice capture directly into the vault.

Media & Files

  • Jellyfin — self-hosted media server
  • Immich — self-hosted photo/video library, drop-in Google Photos alternative
  • Syncthing — continuous cross-device file sync, no cloud intermediary

Dev Tooling

  • Gitea — self-hosted git hosting; primary remote for personal and homelab repositories, integrated with Obsidian Git for automated vault backups

Navigation

  • Homepage.dev — unified dashboard aggregating live status and quick links across every running service
  • Karakeep — self-hosted bookmarking / read-it-later

Key Design Decisions

  • Tailscale + Mullvad exit nodes over self-managed WireGuard/VPN — eliminates key distribution and NAT traversal complexity while retaining WireGuard's performance and security model, with Mullvad's exit-node network handling egress privacy instead of a separately managed VPN client. No ports exposed to the public internet.
  • Caddy over nginx — automatic HTTPS and native Tailscale certificate support reduce the maintenance burden for TLS across services.
  • Gitea over GitHub for private repos — full control over repository data, no external dependency for personal and homelab infrastructure code.
